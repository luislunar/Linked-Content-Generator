# LinkedIn Post Types — Rotation Reference

Source: https://magicpost.in/blog/best-posts-for-linkedin

Use these as the format rotation pool. Daily agent should pick a different type
than the last 3 days. Match type to topic when possible (see "best for" column).

| # | Type | Structure | Length | Best for |
|---|---|---|---|---|
| 1 | **Inspiration / Insight** | Quote or punchy insight + 1 line of personal connection | 50–150 words | Short engagement boost, ideas that need no proof |
| 2 | **Carousel (PDF doc post)** | 5–10 slides: cover → one idea per slide → CTA slide | Minimal text per slide | Breaking down frameworks, lists, multi-step concepts |
| 3 | **Client Success Story** | Context → problem → solution → measurable result | 200–300 words | Credibility, social proof, non-salesy |
| 4 | **Personal Story → Lesson** | Anecdote → tension/turning point → business takeaway | 200–400 words | Humanizing the brand, building trust |
| 5 | **Tips / Best Practices List** | Promise headline → numbered list (5–10 items) → CTA | Scannable bullets | Pure educational value, expertise positioning |
| 6 | **Achievement / Milestone** | The win → the journey → gratitude + lesson | 150–250 words | Launches, anniversaries, milestones |
| 7 | **Industry News + Hot Take** | News in 1–2 lines → contrarian analysis → implication | 150–300 words | Trending topics, thought leadership |
| 8 | **Behind-the-Scenes** | Casual reveal of process → what was learned → invite | 100–200 words | Authenticity, breaks up promotional content |

## Rules
- Never use the same type 2 days in a row.
- Within any 7-day window, hit at least 5 different types.
- Pair Carousel posts with multi-slide PDF (not single image). Other types use single image.
- Default cadence: ~1 carousel every 3-4 days, rest single image.
